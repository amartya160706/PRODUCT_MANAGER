# 📦 Product Manager + jQuery UI Demo

## 📌 Overview

This project is a simple full-stack web application built using Node.js, Express, and MongoDB. It also includes a small frontend demo using jQuery to demonstrate DOM manipulation and animation.

---

## 🚀 Features

### 🔹 Frontend (jQuery Demo)

* Displays multiple paragraphs
* Click on a paragraph to fade it out
* Tracks remaining paragraphs
* Shows alert when all paragraphs are removed

### 🔹 Backend (CRUD Operations)

* Add a product
* Delete a product
* Update product details
* View all products

---

## 🛠️ Tech Stack

* HTML, CSS, JavaScript
* jQuery
* Node.js
* Express.js
* MongoDB
* Mongoose

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Install dependencies:

```
npm install
```

3. Start MongoDB locally

4. Run the server:

```
node index1.js
```

5. Open in browser:

```
http://127.0.0.1:3004/
```

| Method | Route   | Description       |
